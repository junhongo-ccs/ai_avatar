# サービスステーション運営サポートサービス

## 概要

サービスステーション向けに、販売価格、契約単価、入金日、売上、入出金、請求書など日々の経理・販売管理負担を軽減するサービス。

## 対応ニーズ

- 月末の事務負担を軽減したい
- 売上・入出金を早く確認したい
- 請求書作成を効率化したい
