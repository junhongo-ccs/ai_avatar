# ArcGIS

## 概要

地理情報システム（GIS）を用途に応じて構築できるパッケージ。位置情報管理、ルート検索、ラスタ解析、三次元表示、設備管理などに対応する。

## 対応ニーズ

- 位置情報を管理・可視化したい
- 設備や農地情報を地図で共有したい
- ルート検索や空間解析を行いたい

## RAG検索キーワード

ArcGIS, GIS, 地理情報, 位置情報, 設備管理, 農地情報, 空間解析

「ArcGIS」は、地理情報システム（GIS）を幅広い利用形態に
あわせたシステムとして提供するパッケージソフトウェアです。
大量な位置情報の管理、ルート検索等の高度な解析やラスタ

製品群を組み合わせてシステムを構築できます。

析機能などをWebサービスとして共有・公開、一元管理でき

農地情報共有システム
農地情報をGIS上で一元管理。航空写真や背景地図、農地データなどを重ね合わせ、Webで
配信・提供。情報の登録・編集や、農地集積のためのシミュレーションが容易にできます。
通信会社向け設備管理システム
通信設備やサービスエリア情報、通信障害などをGIS上で管理し、Webで情報共有。設備
の場所、サービスエリア、障害範囲を地図上に表示し、正確な位置を容易に把握できます。
災害時情報共有システム
さまざまな情報を地図上に重ね合わせて効果的に可視化し、自然災害やパンデミック
発生時の意思決定を支援するCOP（Common Operation Picture、状況認識の統
一図）を作成。自社設備の被害状況、気象予測情報・推定震度、通行実績情報などの
レイヤ群を組み合わせ、さなざまな状況や組織に応じたCOP 作成に柔軟に対応します。
走行画像アーカイブ／走行指定画像提供サービス
道路走行調査を専門に行う会社が提供する走行画像アーカイブや、オーダーメイドの走
行画像と地図ビューワをセットで提供するサービスです。道路モニタリング調査や都市
計画などへの利用が可能です。
