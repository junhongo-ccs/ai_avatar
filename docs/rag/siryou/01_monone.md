# 異音検知ソリューション Monone

## 概要

製造設備や機械システムが発する音をAIでモデル化し、正常音との差分から異常を検知するソリューション。熟練者の聴き分けに依存していた異音検査や設備状態監視を定量化する。

## 対応ニーズ

- 異音検査を自動化したい
- 音による完成品検査を定量化したい
- 装置の状態を把握したい
