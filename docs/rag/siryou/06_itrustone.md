# iTrustOne

## 概要

金融機関向けに、投信・公共債・保険・外貨など金融商品販売を支援するソフトウェア。顧客属性、アンケート、取引、他システム連携に対応する。

## 対応ニーズ

- 金融商品販売を支援したい
- タブレットで訪問先注文を完結したい
- CRMやネットバンキングと連携したい
