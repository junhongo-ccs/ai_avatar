# NTTデータCCS ソリューション資料 全体概要

## 概要

この資料は、NTTデータCCSの複数のソリューションを紹介する資料です。AI、データサイエンス、RPA、金融機関向け業務支援、サービスステーション運営、グループ経営管理、ITインフラ、GISなどの領域を含みます。

## 想定質問

- NTT DATA CCSの主なサービスソリューションについて教えてください。
- NTTデータCCSの主なサービスソリューションについて教えてください。
- NTTデータCCSにはどのようなソリューションがありますか？
- この資料で紹介されているサービスやソリューションを一覧で教えてください。

## 回答方針

NTT DATA CCSまたはNTTデータCCSの主なサービスソリューションを聞かれた場合は、以下のソリューション一覧を根拠に、カテゴリ別または箇条書き相当の短い文章で要約します。

## ソリューション一覧

| ソリューション | ページ | 主な用途 |
|---|---:|---|
| 異音検知ソリューション Monone | 1-2 | 製造設備や機械システムが発する音をAIでモデル化し、正常音との差分から異常を検知するソリューション。熟練者の聴き分けに依存していた異音検査や設備状態監視を定量化する。 |
| 水稲AI画像解析ソリューション | 3-4 | スマートフォンや固定カメラで撮影した水稲画像から、生育ステージ、茎数、雑草、病害虫をAIで診断し、追肥時期や防除策の判断を支援するソリューション。 |
| データサイエンスソリューション | 5-6 | モニタリング、可視化、シミュレーション、分析の技術と地球科学分野の専門知識を組み合わせ、地理情報、画像処理AI、ビッグデータ、地球科学・資源の4領域を提供するソリューション。 |
| UXサイクルソリューション | 7-8 | 顧客接点の企画、UIデザイン、PoC検証を通じてUX向上を支援し、サービス検討とプロダクト開発のサイクルを連携させるソリューション。 |
| RPAソリューション WinActor | 9-10 | 定型作業をRPAで自動化し、業務アプリケーション間のデータ受け渡しや登録・確認作業などを効率化するソリューション。 |
| iTrustOne | 11-12 | 金融機関向けに、投信・公共債・保険・外貨など金融商品販売を支援するソフトウェア。顧客属性、アンケート、取引、他システム連携に対応する。 |
| サービスステーション運営サポートサービス | 13-14 | サービスステーション向けに、販売価格、契約単価、入金日、売上、入出金、請求書など日々の経理・販売管理負担を軽減するサービス。 |
| GROUP CATS | 15-16 | グループ各社の会計情報・財務情報を集約・一元管理し、グループ経営の課題解決を支援するサービス型ソリューション。 |
| ITインフラ運用・支援サービス | 17-18 | ENEOSグループで培ったノウハウを活かし、設計・構築後のITインフラ運用保守や個別運用作業を支援するサービス。 |
| ITインフラ構築サービス | 19-20 | 上流SEやITアーキテクトが、仮想基盤、データセンター、キャリア選定などITインフラ構築をコンサルティングから支援するサービス。 |
| ArcGIS | 21-22 | 地理情報システム（GIS）を用途に応じて構築できるパッケージ。位置情報管理、ルート検索、ラスタ解析、三次元表示、設備管理などに対応する。 |
| geoassist | 23-24 | 誰でも簡単に地理情報を用いた業務効率化と意思決定支援を実現するGISフレームワーク。顧客情報、車両位置、安否情報などを地図で共有できる。 |
